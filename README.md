# 🎵 Beatflare Music Player

<img src="./images/art.png" width="150"/>

A modern, sleek, and open-source music player built with love by [Bahasuru Nayanakantha](https://www.linkedin.com/in/bahasuru-nayanakantha/). Beatflare brings your favorite tracks to life with stylish UI, advanced features like Party Mode, a customizable equalizer, and seamless song management.


## 🖼️ Preview

![image](https://github.com/user-attachments/assets/8c231e55-41ef-4654-81c0-3e838fce64ec)

## 🚀 Features

- ▶️ **Play, Pause, and Skip** tracks effortlessly  
- 📂 **Supports multiple audio formats**  
- 🌙 **Light & Dark Mode** for any mood  
- 🎚️ **Customizable Equalizer**  
- 📃 **Effortless Song Management**  
- 🎉 **Surprise Party Mode!**


## 📦 Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/bahasuru-naya/Beatflare-Music-Player.git
2. **Navigate to the project folder**
    ```bash
   cd Beatflare-Music-Player    
3. **Open `index.html` in your browser**
